# Contributing

Thanks for helping improve MiniMCP!

## Quick Steps

1. Fork the repo and create a branch
2. Make your changes
3. Run checks:
   ```bash
   deno task fmt
   deno task lint
   deno task check
   deno task test
   ```
4. Open a pull request

## Guidelines

- Keep it simple — this is a learning project
- Follow existing code style (TypeScript strict, 2-space indent)
- Add comments only where they clarify intent
- Update docs if your change affects usage
- No unnecessary dependencies

## Good Contributions

- Bug fixes
- Clearer documentation
- Better error messages
- Simple, educational examples

## Code of Conduct

Be respectful and constructive. This is a learning community.
